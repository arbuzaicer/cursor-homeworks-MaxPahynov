Перелік основних команд Git

  • git config --global user.name blabla - встановлення назви користувача для GIT bash

  • git config --global user.email blabal@bla - пошта користувача для GIT bash

  • git init - ініціалізація локального репозиторія

  • git status - перевірка стану файлів на момент відправки на віддалений репозиторій

  • git add - додавання відстеження змін файлів проекту

  • git commit - коміт проекту

  • git log - вивід списку логів/комітів

  • git checkout - перехід до попередньої версії проекту по хешу (ідентифікатору коміту)

  • git branch - вибір вітки проекту (dev/master etc)

  • git push - завантаження проекту на репозиторій

  • git pull - завантаження проекту на локальний ПК з репозиторію

  • git merge - "склеювання" віток

  • git clone - клонування проекту по HTTPS || SSH

  • .gitignore - файл, який вказує які файли / папки не потрібно завантажувати в віддалений репозиторій
